# 星辰跳动 - 我的个人博客

星辰跳动使用「Stellar」主题。Stellar 是一个内置文档系统的简约商务风 Hexo 主题，支持丰富的标签和动态数据组件。


## Getting Started

检查你的环境:

```yaml
Hexo: 6.3.0 ~ 7.3.0
hexo-cli: 4.3.0 ~ latest
node: 14.17.3 ～ 22.12.0 # 建议选择 LTS 版本。
npm: 6.14.13 ~ 10.2.3
```

编辑你的 `_config.yml`:

```yaml
theme: stellar
```

在终端中安装 Stellar:

```bash
npm i
npm i hexo-theme-stellar
```
